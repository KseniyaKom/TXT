TXT
 1. github.com, click + --> New repository -cоздать внешний репозиторий c названием TXT.
 2. git clone (ссылка на репозиторий) - клонировать репозиторий TXT на локальный компьютер.
 3. cd TXT / cat > new.txt / ctrl+c - внутри локального TXT создать файл “new.txt”.
 4. git add "new.txt" - добавить файл под гит.
 5. git commit -m "new.txt" - закоммитить файл.
 6. git push - отправить файл на внешний GitHub репозиторий.
 7. vim "new.txt" / insert / esc / :wq - отредактировать содержание файла “new.txt” - написать информацию о себе (ФИО, возраст, количество домашних животных, будущая желаемая зарплата). Всё написать в формате TXT.
 8. git add "new.txt" / git commit -m "new.txt" / git push - отправить изменения на внешний репозиторий.
 9. cat > preferences.txt / ctrl+c - создать файл preferences.txt
 10. vim "preferences.txt" / insert / esc / :wq - в файл preferences.txt” добавить информацию о своих предпочтениях (Любимый фильм, любимый сериал, любимая еда, любимое время года, сторона которую хотели бы посетить) в формате TXT.
 11. vim "skills.txt" / insert / esc / :wq - cоздать файл sklls.txt добавить информацию о скиллах которые будут изучены на курсе в формате TXT
 12. git add . ; git commit -m "prefernces/skills" - cделать коммит в одну строку.
 13. git push - отправить сразу 2 файла на внешний репозиторий.
 14. Add new file/Create new file - на веб интерфейсе создать файл bug_report.txt.
 15. Commit new file - сделать Commit changes (сохранить) изменения на веб интерфейсе.
 16. Edit this file - на веб интерфейсе модифицировать файл bug_report.txt, добавить баг репорт в формате TXT.
 17. Commit changes - сделать Commit changes (сохранить) изменения на веб интерфейсе.
 18. git pull - синхронизировать внешний и локальный репозиторий TXT
